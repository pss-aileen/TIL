### サーバーとは
[サーバーとは？｜サーバーとは何か、サーバーの種類や仕組みなどを3分でわかりやすく解説します - YouTube](https://www.youtube.com/watch?v=0EI_lmrijQs)

- サーバーとは
  - サービスや機能を提供するコンピューターのこと
- サーバーは2つある
  - 集中システム
    - 1台のPCが全ての処理をする
  - 分散システム
    - 複数のコンピューターで処理を分けておこなう
    - サーバー負荷を軽減できる
    - 一部が故障してもシステムがダウンしない
- サーバーの種類
  - ファイルサーバー
  - プリントサーバー
  - メールサーバー
