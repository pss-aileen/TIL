- Vue Documents
  - Introduction: done
    - original:
      - When you want to declare variables as "reactive", you set these `reactive()`, `ref()`.
      - `reactive()` is like Objects.
      - `ref()` is just one value.
    - modified:
      - To declare variables as "reactive", you use `reactive()` or `ref()`
      - `reactive()` works like objects.
      - `ref()` holds a single value.
    - modified:
      - `reactive()`: Makes the entire objects reactive, including nested properties, allowing Vue to track chnages at all level.
      - `ref()`: Makes a single value or non-object type reactive. When wrapping an object with `ref()`, some limitations exist, as nested properties won't be autimatically reactive.
